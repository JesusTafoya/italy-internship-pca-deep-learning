# PCA & Deep Learning: Multi-Dataset Comparison

Comparative analysis of three datasets using Principal Component Analysis (PCA) combined with deep learning, developed during a research internship in Italy (2020).

## Overview

This project explores dimensionality reduction and pattern discovery across three distinct datasets, using PCA to reduce feature space and deep learning models to analyze and compare the resulting structures. The goal was to evaluate how well PCA-transformed features supported downstream deep learning performance across the datasets.

## Approach

- Preprocessed and standardized three separate datasets
- Applied PCA to reduce dimensionality and identify principal components
- Trained deep learning models on PCA-transformed data
- Compared results across datasets to evaluate consistency, performance, and generalizability of the approach
